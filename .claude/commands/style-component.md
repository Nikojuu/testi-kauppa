# Style Component Command

Style or create a component following the Pupun Korvat design system.

## Instructions

First, read the styling guidelines:
1. Read `.claude/skills/pupun-korvat-styling.md` for the complete design system
2. Read `.claude/agents/styling-agent.md` for implementation patterns

Then apply the Pupun Korvat styling to the requested component using:
- Rose gold & champagne color palette
- Corner accent decorations
- Diamond shape motifs
- Smooth Framer Motion animations
- Elegant typography with proper font families
- Gradient overlays and lines

Always ensure the result feels like a luxury artisan jewelry brand.

Component to style: $ARGUMENTS
